# @pcl-community/plain-ui-react-tailwind-ts

Plain UI - React + TailwindCSS + TypeScript 组件库

## 安装

```bash
npm install @pcl-community/plain-ui-react-tailwind-ts
# 或
yarn add @pcl-community/plain-ui-react-tailwind-ts
# 或
pnpm add @pcl-community/plain-ui-react-tailwind-ts
```

## 使用

### 完整引入

```typescript
import { Button } from '@pcl-community/plain-ui-react-tailwind-ts'
import '@pcl-community/plain-ui-react-tailwind-ts/style.css'

function App() {
  return <Button>Click me</Button>
}
```

## 技术栈

- React 18+
- TailwindCSS 4.0+
- TypeScript 5.4+
