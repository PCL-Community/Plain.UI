# @pcl-community/plain-ui-react-tailwind-js

Plain UI - React + TailwindCSS + JavaScript 组件库

## 安装

```bash
npm install @pcl-community/plain-ui-react-tailwind-js
# 或
yarn add @pcl-community/plain-ui-react-tailwind-js
# 或
pnpm add @pcl-community/plain-ui-react-tailwind-js
```

## 使用

```javascript
import { Button } from '@pcl-community/plain-ui-react-tailwind-js'
import '@pcl-community/plain-ui-react-tailwind-js/style.css'

function App() {
  return <Button>Click me</Button>
}
```

## 技术栈

- React 18+
- TailwindCSS 4.0+
- JavaScript
