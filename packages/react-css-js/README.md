# @pcl-community/plain-ui-react-css-js

Plain UI - React + CSS + JavaScript 组件库

## 安装

```bash
npm install @pcl-community/plain-ui-react-css-js
# 或
yarn add @pcl-community/plain-ui-react-css-js
# 或
pnpm add @pcl-community/plain-ui-react-css-js
```

## 使用

```javascript
import { Button } from '@pcl-community/plain-ui-react-css-js'
import '@pcl-community/plain-ui-react-css-js/style.css'

function App() {
  return <Button>Click me</Button>
}
```

## 技术栈

- React 18+
- CSS
- JavaScript
