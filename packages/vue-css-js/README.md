# @pcl-community/plain-ui-vue-css-js

Plain UI - Vue 3 + CSS + JavaScript 组件库

## 安装

```bash
npm install @pcl-community/plain-ui-vue-css-js
# 或
yarn add @pcl-community/plain-ui-vue-css-js
# 或
pnpm add @pcl-community/plain-ui-vue-css-js
```

## 使用

### 完整引入

```javascript
import { createApp } from 'vue'
import PlainUI from '@pcl-community/plain-ui-vue-css-js'
import '@pcl-community/plain-ui-vue-css-js/style.css'

const app = createApp(App)
app.use(PlainUI)
```

### 按需引入

```javascript
import { Button } from '@pcl-community/plain-ui-vue-css-js'
import '@pcl-community/plain-ui-vue-css-js/style.css'
```

## 技术栈

- Vue 3.4+
- CSS
