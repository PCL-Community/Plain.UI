# @pcl-community/plain-ui-react-css-ts

Plain UI - React + CSS + TypeScript 组件库

## 安装

```bash
npm install @pcl-community/plain-ui-react-css-ts
# 或
yarn add @pcl-community/plain-ui-react-css-ts
# 或
pnpm add @pcl-community/plain-ui-react-css-ts
```

## 使用

### 完整引入

```typescript
import { Button } from '@pcl-community/plain-ui-react-css-ts'
import '@pcl-community/plain-ui-react-css-ts/style.css'

function App() {
  return <Button>Click me</Button>
}
```

## 技术栈

- React 18+
- CSS
- TypeScript 5.4+
